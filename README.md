<ul>
  <li>создание схемы</li>
  <li>разграничение прав доступа</li>
  <li>колонки</li>
  <li>связи</li>
   <li>определение уникальных ограничений</li>
  <li>наложение индексов на колонки</li>
  <li>триггеры для автоматизации некоторых процессов</li>
</ul>
